# 業務ルール・間引き条件

このファイルにルールを追記していくことで、AIの精度が上がります。

---

## 間引きルール（スケジュールから除外するもの）

### 自動除外（アプリが自動で処理）
- 最終列が「無」の行
- 担当者表で「長瀬」以外が担当のホール
- 「景品入荷」「クロロ景品」などの景品系取材
- 灰色行（キャンセル）→ 手動でコピペ前に除外してください

### 除外キーワード
※ アプリのマスタタブ→間引きルールからも設定できます

```
（ここに除外したいホール名・企業名などを記入）
例：
アビバ
プロテラス
ガーデン
```

### 除外条件
```
（ここに条件を記入）
例：
・URLが含まれていない行は除外
・担当者名が「大薮」以外は除外
```

---

## エラー検知ルール

- 遊技人数が500名を超える → 桁ミスの可能性
- 同じホール名から2回以上報告 → 重複の可能性
- 当日スケジュールにないホール名 → スケジュール外報告

---

## 担当者ルール

- 担当者表で「長瀬」が担当するホールは**長瀬担当タブ**に表示
- 長瀬担当のポストは「完了ホール（全体）」にカウント（ポスト済み自分にはカウントしない）
- 報酬計算はポスト生成タブのポスト数で行う
