# 取材ポストジェネレーター

パチンコホール来店取材の広告業務用Xポスト文自動生成アプリ。

---

## 🚀 使い方（GitHub Pages）

1. GitHubで新しいリポジトリを作成
2. `index.html` をアップロード
3. Settings → Pages → Branch: main → Save
4. `https://ユーザー名.github.io/リポジトリ名/` でアクセス

### スマホのホーム画面に追加
- **iPhone**: Safariで開く → 共有ボタン（□↑）→「ホーム画面に追加」
- **Android**: Chromeで開く → ⋮ →「ホーム画面に追加」

---

## 📱 機能一覧

### ① スケジュール登録（朝）
- **当日スケジュール**（取材名・ホール名・URL列）をコピペ
- **各取材担当者表**（ホール名・都道府県・企業名・担当者）をコピペ
- AIが自動で間引き・担当者フィルタリング
- **新規登録**：リストをリセット／**追加登録**：追記（重複スキップ）
- 最終列「無」の行は自動除外
- 「長瀬」以外の担当者ホールは自動除外
- マスタ未登録の取材名はモーダルで「除外」or「新規登録」を選択

### ② ポスト生成（昼〜夜）
- LINEの報告文をそのまま貼り付け
- AIが自動解析：ホール特定・人数抽出・機種抽出
- エラー検知：桁ミス・スケジュール外ホール・重複報告
- 手動編集（人数・機種）→ポスト文再生成

### ③ 進捗管理
- ✍️ ポスト済み（自分）：LINEポスト生成タブでポスト済みにしたもの
- ✅ 完了ホール（全体）：自分＋他担当者含め確認済みのもの
- ⚠️ 未報告ホール：まだ完了していないホール
- 未報告ホールをまとめてLINEにコピーして問い合わせ可能

### ④ 長瀬担当
- 担当者表で「長瀬」が担当するホールを自動抽出
- 当日スケジュールと照合して取材名を紐付け
- 同一ホールで2取材同時開催の場合は別々に表示
- 現地で人数入力・機種入力→ポスト文生成
- 完了は「完了ホール（全体）」にカウント

### ⑤ マスタ管理
- 取材名・表示名（絵文字）・ジャンル登録
- 絵文字ピッカー（使用頻度順・⚡先頭固定）
- 間引きルール設定

---

## ⚙️ ルール・仕様

### 間引きルール
- 最終列「無」→ 除外
- 担当者表で「長瀬」以外の担当者 → 除外
- 「景品入荷」系取材 → 除外
- 灰色行（キャンセル）→ 手動で除いてからコピペ

### 時間帯判定
- 〜16:00 → ☀️ 通常テンプレート
- 16:00〜21:00 → 🌆 夕方テンプレート（同上）
- 21:00〜 → 🌙 夜テンプレート（「盛り上がっていたようですね」「PR」のみ）

### 稼働人数変換
実数 → 公表人数（10人単位で切り上げ）
440人超 → 上限500人

### 高稼働機種
- 最大5機種
- 合計40文字制限（超えた分は自動省略）

### ポストテンプレート（通常・夕方）
```
M/D [取材表示名] [ホール名]

現場の状況をリポートします📸
スロットコーナーの稼働約XX名

📝高稼働機種
・機種1
・機種2

店内盛り上がってるようですね⤴️⤴️
この後も、まだまだチャンスがありそう👍

PR
```

### ポストテンプレート（夜）
```
M/D [取材表示名] [ホール名]

現場の状況をリポートします📸
スロットコーナーのピーク時稼働約XX名

📝高稼働機種
・機種1

店内盛り上がっていたようですね⤴️⤴️

PR
```
