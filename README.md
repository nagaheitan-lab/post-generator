# 取材ポストジェネレーター

パチンコホール来店取材の広告業務用Xポスト文自動生成アプリ。

---

## 🚀 使い方（GitHub Pages）

1. GitHubで新しいリポジトリを作成
2. `index.html` をアップロード
3. Settings → Pages → Branch: main → Save
4. `https://ユーザー名.github.io/リポジトリ名/` でアクセス

---

## 📱 機能一覧

### ① スケジュール登録（朝）
- 作業タブA列（取材名）・B列（ホール名）をコピペ
- AIが本業務分を自動間引き・取材マスタと照合
- 当日リストとして記憶

### ② ポスト生成（昼〜夜）
- LINEの報告文をそのまま貼り付け
- AIが自動解析：ホール特定・人数抽出・機種抽出
- エラー検知：桁ミス・スケジュール外ホール・重複報告

### ③ 進捗管理
- 実施済み／未実施リスト
- 当日ポスト数カウント
- 夜の漏れ防止

### ④ マスタ管理
- 取材名・表示名（絵文字）・ジャンル登録
- 間引きルール設定

---

## ⚙️ ルール・仕様

### 時間帯判定
- 〜16:00 → ☀️ 通常テンプレート（「盛り上がってるようですね」「まだまだチャンスがありそう」）
- 16:00〜21:00 → 🌆 夕方テンプレート（同上）
- 21:00〜 → 🌙 夜テンプレート（「盛り上がっていたようですね」「PR」のみ）

### 稼働人数変換（早見表）
実数 → 公表人数（10人単位で切り上げ）
440人超 → 上限500人

### 高稼働機種
- 最大5機種
- 合計40文字制限（超えた分は自動省略）

### ポストテンプレート（通常）
```
M/D [取材表示名] [ホール名]

現場の状況をリポートします📸
スロットコーナーの稼働約XX名

📝高稼働機種
・機種1
・機種2

店内盛り上がってるようですね⤴️⤴️
この後も、まだまだチャンスがありそう👍

PR
```

### ポストテンプレート（夜）
```
M/D [取材表示名] [ホール名]

現場の状況をリポートします📸
スロットコーナーのピーク時稼働約XX名

📝高稼働機種
・機種1

店内盛り上がっていたようですね⤴️⤴️

PR
```
